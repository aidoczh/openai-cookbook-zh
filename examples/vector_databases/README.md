# 向量数据库

本节《OpenAI Cookbook》展示了多种可用于支持语义搜索应用场景的向量数据库。

向量数据库对于知识检索应用来说是一个极佳的辅助工具，它通过为大型语言模型（LLM）提供相关上下文来回答问题，从而减少幻觉现象。

每个提供商都有自己的命名目录，其中包含一个标准笔记本，引导您如何使用我们的API与他们的产品结合，以及他们选择添加的任何补充笔记本，以展示其功能。
